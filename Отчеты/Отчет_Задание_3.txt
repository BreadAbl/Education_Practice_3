ОТЧЁТ ПО ЗАДАНИЮ 3
Тема: Доработка программного решения и создание сопутствующей документации

ТЕХНИЧЕСКОЕ РУКОВОДСТВО
Подготовлен документ «Руководство системного программиста 01»,
содержащий описание архитектурных решений, системных требований,
последовательности установки библиотек, инструкции по запуску Flask-приложения,
настройке конфигурационных параметров и переменных среды.
Документ включает сведения о структуре проекта, организации модулей, работе с СУБД, процедурах резервирования, обновления версий и мерах информационной безопасности.

ВЫПОЛНЕНИЕ ТРЕБОВАНИЙ ПО РАСШИРЕНИЮ ФУНКЦИОНАЛА
В систему интегрирована роль «Менеджер по качеству» с правами просмотра всех обращений,
назначения и перераспределения исполнителей, мониторинга сроков и анализа статистики заявок.
Программный код дополнен проверками типа учётной записи, интерфейсы адаптированы для отображения новых возможностей.

Согласно дополненному техническому заданию реализован механизм создания QR-кодов со ссылками на Google-форму для сбора клиентских отзывов о работе сервисного центра.
QR-код отображается в интерфейсе после закрытия заявки и даёт возможность оценить качество обслуживания.

РЕКОМЕНДАЦИИ ПО РАЗВИТИЮ СИСТЕМЫ
Составлен перечень предложений по улучшению функционала. В него вошли следующие направления:
- внедрение email и SMS-информирования о ходе выполнения заявок;
- добавление календаря для планирования выездов специалистов;
- создание мобильного клиента для мастеров;
- расширенная панель аналитики с визуализацией данных;
- интеграция с внешними CRM-платформами.

Для каждого направления определены приоритетность, оценка трудоёмкости и примерные временные рамки реализации.

АУДИТ КАЧЕСТВА ИСХОДНОГО КОДА
Выполнен анализ кода по параметрам: полнота обработки исключительных ситуаций, наличие автоматизированных тестов,
проверка вводимых данных, механизмы восстановления после сбоев, документирование ключевых функций.
Сформирован отчёт с оценками и предложениями по совершенствованию: внедрение единой системы журналирования, расширение покрытия автотестами, усиление механизмов безопасности и улучшение комментариев в сложных фрагментах кода.

ИТОГИ
Все требования третьего задания реализованы: разработано руководство для системных специалистов,
добавлена новая роль пользователя и функция генерации QR-кодов, подготовлены рекомендации по дальнейшему развитию и проведена оценка качества программного кода.
